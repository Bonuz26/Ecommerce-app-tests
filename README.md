# Ecommerce Platform Test

### 1. Klonowanie repozytorium

```bash
git clone https://github.com/Bonuz26/Ecommerce-app-test.git
cd Ecommerce-app-test
```

### 2. Instalacja zależności

```bash
npm install
```

### 3. Uruchomienie aplikacji

```bash
npm run dev
```

Aplikacja będzie dostępna pod adresem [http://localhost:5173](http://localhost:5173).

### 4. Uruchamianie testów E2E z UI

```bash
npm run test:e2e:ui
```

### 5. Uruchamianie testów jednostkowych i mock (Vitest UI)

```bash
npm run test:ui
```

## Technologie użyte

- React
- Redux
- Context API
- Tailwind CSS
- FakeStoreAPI
- React Router DOM
- Formik
- Yup
- SweetAlert
- Vitest
- Playwright
